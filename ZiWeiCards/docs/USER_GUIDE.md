# 紫微牌卡 使用者操作指南

## 簡介

**紫微牌卡**是由沐心紫微提供的線上占卜工具，讓您透過抽取紫微牌卡來獲得指引與啟示。本指南將引導您完成所有功能的操作。

---

## 前置條件

- 網路連線（建議使用 Wi-Fi 或穩定的行動網路）
- 支援的瀏覽器：Chrome、Safari、Edge 或 Firefox（建議使用最新版本）
- 裝置：電腦、平板或手機皆可使用

---

## 功能一覽

| 功能     | 說明                               |
| -------- | ---------------------------------- |
| 擲筊     | 虛擬擲筊，獲得聖杯、笑杯或蓋杯結果 |
| 抽牌     | 從牌庫中隨機抽取牌卡               |
| 洗牌     | 重新洗混牌庫，準備下一次抽牌       |
| 儲存牌面 | 將目前顯示的牌面儲存為圖片         |

---

## 操作步驟

### 步驟一：開啟應用程式

1. 在瀏覽器中開啟紫微牌卡網頁。
2. 畫面載入完成後，您會看到控制區與牌卡顯示區。

---

### 步驟二：輸入問題（選填）

1. 在「**輸入問題**」欄位中輸入您想詢問的問題。
2. 儲存牌面時，問題會自動顯示在圖片上。

> [!TIP]
> 問題輸入為選填，若不輸入問題也可以直接抽牌。

---

### 步驟三：選擇牌陣

1. 點擊「**選擇牌陣**」下拉選單。
2. 從以下選項中選擇一種牌陣：

| 牌陣名稱           | 牌卡組成                   | 適用情境             |
| ------------------ | -------------------------- | -------------------- |
| 【擲筊】           | 兩枚筊                     | 簡單是非問題         |
| 【一張】牌陣       | 1 張長生牌                 | 快速獲得能量指引     |
| 【二張】牌陣       | 1 主星 + 1 輔星            | 簡易問題分析         |
| 【三張】基礎牌陣   | 1 主星 + 1 輔星 + 1 長生   | 一般問題諮詢（預設） |
| 【對宮】牌陣       | 2 主星 + 2 輔星 + 1 長生   | 對比分析、二選一問題 |
| 【三方四正】牌陣   | 4 主星 + 4 輔星 + 1 長生   | 多面向深度分析       |
| 【十二宮位】大牌陣 | 13 主星 + 12 輔星 + 1 長生 | 完整命盤式解讀       |

---

### 步驟四：抽牌 / 擲筊

1. 點擊「**抽牌**」按鈕（若選擇擲筊，按鈕會顯示「擲筊」）。
2. 牌卡會先顯示背面並進行洗牌動畫。
3. 動畫完成後，牌卡翻轉顯示結果。

**預期結果：**

- 畫面顯示抽取的牌卡（正牌或倒牌）。
- 若選擇擲筊，會顯示兩枚筊的結果：聖杯、笑杯或蓋杯。
- 上方顯示「抽牌時間」或「擲筊時間」。

---

### 步驟五：洗牌 / 重來

1. 若想重新抽牌，點擊「**洗牌**」按鈕。
2. 牌卡會恢復背面狀態，準備下一次抽牌。

> [!NOTE]
> 若選擇擲筊模式，按鈕會顯示「重來」，功能相同。

---

### 步驟六：儲存牌面

1. 抽牌完成後，點擊「**儲存牌面**」按鈕。
2. 畫面會顯示「儲存中...」的讀取動畫。
3. 儲存完成後，系統會自動下載圖片。

**預期結果：**

- 圖片檔案名稱格式：`紫微牌卡_[牌陣名稱]_[問題]_[時間].png`
- 圖片包含：牌面、問題文字（如有輸入）、時間戳記。

> [!IMPORTANT]
> 在儲存過程中，請勿重複點擊按鈕，系統會自動處理。

## 牌卡類型說明

本應用程式使用三種類型的牌卡：

| 牌卡類型 | 數量              | 說明                       |
| -------- | ----------------- | -------------------------- |
| 主星牌   | 78 張（含正倒牌） | 紫微斗數主星，包含十四主星 |
| 輔星牌   | 24 張（含正倒牌） | 輔助星曜，提供補充訊息     |
| 長生牌   | 24 張（含正倒牌） | 長生十二神，代表運勢階段   |

> [!NOTE]
> 每張牌卡皆有「正牌」與「倒牌」兩種狀態，代表不同的能量表現。

## 行動裝置操作提示

### iPhone / iPad

- 儲存牌面後，圖片會出現在「分享」選單中。
- 選擇「儲存圖片」可將圖片存入相簿。

### Android

- 儲存牌面後，圖片會自動下載至「下載」資料夾。
- 若出現分享選單，可選擇儲存或分享至其他應用程式。

### 加入主畫面

1. 在瀏覽器中開啟紫微牌卡網頁。
2. **iPhone**：點擊分享按鈕 → 選擇「加入主畫面」。
3. **Android**：點擊選單按鈕 → 選擇「新增至主畫面」。

## 疑難排解

### 問題：牌卡沒有顯示

**解決方法：**

1. 檢查網路連線是否正常。
2. 重新整理頁面（按 F5 或下拉刷新）。
3. 清除瀏覽器快取後再試。

### 問題：儲存牌面後沒有反應

**解決方法：**

1. 等待數秒，儲存過程需要時間。
2. 確認瀏覽器允許下載檔案。
3. 在手機上，檢查「下載」資料夾或相簿。

### 問題：圖片品質不佳

**解決方法：**

1. 使用較大螢幕的裝置（如平板或電腦）。
2. 確保完整載入後再儲存。
